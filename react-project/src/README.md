
1. What is JSX, and why is it used?
JSX holo JavaScript-er bhetore HTML lekhar ekta style.
Keno dorkar? Eta diye khub sohoje UI banano jay. HTML-er bhetorei {} bracket use kore JavaScript-er logic (jemon: calculation ba variable) likha jay.

2. What is the difference between State and Props?
Props holo ekti component-e bahir theke (Parent theke) pathano data, ja shudhu pora jay  kintu change kora jay na.
State holo ekti component-er nijer bhetorer data, ja dorkar moto update kora jay ebong change hole UI eh change hoy.

3. What is the useState hook, and how does it work?
useState holo React-er ekta function, ja diye component er bhetore kono data rakha jay.
Kivabe kaj kore? Eta theke 2ta jinish pawa jay ekta holo Variable data rakhar jonno ar ekta holo Function.Data change holei React automatically screen update kore dey.

4. How can you share state between components in React?
React-e data upor theke niche jay. Jodi duiti component-er bhetore state share korte hoy
Lifting State Up: State-ke tader common "Parent" component-e niye jete hoy. Erpor Props-er madhyome niche pathate hoy.
Context API: Jodi onek niche data pathate hoy, tokhon Context use kora hoy jate bar bar Props pass korte na hoy.
